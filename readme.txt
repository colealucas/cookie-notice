== Description ==
Create Cookie Law Notification for your WordPress website. The plugin allows you to write a 
cookie alert message, provide links and two buttons for user interaction. You can customize 
colors and cookie bar position.


== Features ==
* 3 color themes and 4 positions to choose from
* Cross browser compatible
* Smooth show in animation
* Quick Installation and Simple Configuration
* Powered by Pure Javascript


== Installation ==
1. Install and activate the plugin on the WordPress Plugins page
2. Access the plugins settings page on the left side wordpress menu 'Cookie Notice'
3. Enable and Configure your cookie notice bar, set position, color theme and customize cookie notice text and buttons.
4. Save changes
4. Enjoy using iComply Cookie Notice :)
